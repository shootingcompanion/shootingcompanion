Plotting Made Simple

Sessions List

The first screen you see in the application. Table on left will display your previous sessions along with date, target type, and your short description entered on the session info screen.

Tapping '+' in the top right will create a new session.

Selecting an existing session in the list will let you see additional details on the right and edit the session with the button in the bottom right.

The 'share' button (top right next to '+' button) allows you to export the data for all saved sessions

Handy Tips:

  • Swipe left on a row in the table to delete a session
  • Tapping an existing session will show the elevation value for the last shot - Handy when you're shooting that distance again

    • Session Info

    Here you can set the date and time of the session, the target face and distance as well as two different text fields to use how you want.

    Tapping 'Continue' in the top right will take you to the main plotting screen.

    The 'share' button (top right next to '+' button) allows you to export the data for just this selected session
